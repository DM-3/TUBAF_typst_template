# TUBAF_typst_template
Inofficial template for writing academic papers and reports for the TU Bergakademie Freiberg using Typst.


# Typographic "Features"
- white paper, DIN A4
- (correct?) margins
    - left: 40 mm
    - top: 25 mm
    - bottom: 20 mm
    - right: 25 mm
- continuous page numbering:
    - left-right aligned footer
    - hidden for cover page and declaration of authenticity
    - roman numerals for outline pages, starting with I on the cover page
    - arabic numerals for content pages, starting with 1 on the first content page
- font:
    - arial
    - 10 pt size for default text
    - 1.5 line spacing
    - hyphenation 
- headings:
    - level 1 headings start a new page
    - gap of 2.5 lines before a heading
    - gap of 1 line after a heading
- outlines:
    - list of contents
    - list of figures
    - list of tables
    - custom (e.g. list of symbols and abbreviations)
